# Mini game Secret Word

Secret Word é um mini game onde o objetivo do usuário é acertar a palavra. Ele terá 3 tentativas e para cada palavra ele terá uma dica sobre a palavra.

## Linguagens e bibliotecas utilizadas

- React
- Hooks (useState, useEffect e useCalback)
- Css3 

#### Criei esse mini game para fins de estudos, onde me aprofundei em React, aprende mais sobre Hooks na utilização do useState, useEffetc e useCalback.
